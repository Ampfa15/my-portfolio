# Portfolio (Static)

This is a simple portfolio website built with **HTML + CSS + JavaScript**.

## Run locally

- Open `index.html` in your browser, or
- Use a local server (recommended). For example (PowerShell):

```powershell
cd "C:\Users\peendu\Downloads\my-portfolio-main\my-portfolio-main"
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize

- **Your name + bio**: edit `index.html`
- **Projects**: edit the `projects` array in `script.js`
- **Skills**: edit the `skills` array in `script.js`
- **Contact email**: replace `yourname@example.com` in `script.js`
